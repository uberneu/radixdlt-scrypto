pub struct NibblePath {
    num_nibbles: usize,
    bytes: Vec<u8>,
}
Expand description

NibblePath defines a path in Merkle tree in the unit of nibble (4 bits).

Fields§

§num_nibbles: usize

Indicates the total number of nibbles in bytes. Either bytes.len() * 2 - 1 or bytes.len() * 2.

§bytes: Vec<u8>

The underlying bytes that stores the path, 2 nibbles per byte. If the number of nibbles is odd, the second half of the last byte must be 0.
