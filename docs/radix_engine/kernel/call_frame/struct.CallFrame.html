pub struct CallFrame<C, L> {
    depth: usize,
    call_frame_data: C,
    owned_root_nodes: IndexSet<NodeId>,
    transient_references: NonIterMap<NodeId, TransientReference>,
    stable_references: BTreeMap<NodeId, StableReferenceType>,
    next_handle: SubstateHandle,
    open_substates: IndexMap<SubstateHandle, OpenedSubstate<L>>,
}
Expand description

A call frame is the basic unit that forms a transaction call stack, which keeps track of the owned objects and references by this function.

Fields§

§depth: usize

The frame id

§call_frame_data: C

Call frame system layer data

§owned_root_nodes: IndexSet<NodeId>

Owned nodes which by definition must live on heap

§transient_references: NonIterMap<NodeId, TransientReference>

References to non-GLOBAL nodes, obtained from substate loading, ref counted. These references may NOT be passed between call frames as arguments

§stable_references: BTreeMap<NodeId, StableReferenceType>

Stable references points to nodes in track, which can’t moved/deleted. Current two types: GLOBAL (root, stored) and DirectAccess. These references MAY be passed between call frames

§next_handle: SubstateHandle§open_substates: IndexMap<SubstateHandle, OpenedSubstate<L>>
