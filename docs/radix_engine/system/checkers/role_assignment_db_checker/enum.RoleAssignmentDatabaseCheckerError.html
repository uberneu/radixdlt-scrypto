    InvalidAccessRule(AccessRule, RoleAssignmentError),

        maximum_length: usize,
        actual: usize,

        role_key: ModuleRoleKey,
    },
}
Expand description

An enum of all of the errors that we may encounter when doing a database check. These errors are collected and then returned at the end of database check.
